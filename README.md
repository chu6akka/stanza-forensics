# Stanza Forensics GUI

Простая программа с графическим интерфейсом (Tkinter) для судебно-лингвистического анализа русского текста.

## Актуальная логика backend

Теперь снова основной backend — **Stanza**.

Порядок резервирования:
1. `stanza` (основной)
2. `natasha` (резерв №1)
3. `pymorphy3` (резерв №2)

Если Stanza в текущем запуске не инициализируется, приложение автоматически переключится на резерв и в интерфейсе покажет фактически используемый backend.

## Что есть в приложении

- переключатель режима: `stanza` / `natasha` / `pymorphy3`;
- русскоязычные морфологические признаки для удобства лингвоюридических специалистов;
- подсветка слова в исходном тексте при наведении на строку таблицы;
- круговая диаграмма долей частей речи;
- частотные коэффициенты POS и базовые автороведческие показатели.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск

```bash
python pos_forensics.py
```
