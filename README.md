# Stanza Forensics

CLI-инструмент для морфологического и синтаксического разбора текста с упором на **специфические части речи**, контекст и признаки слов.

## Что умеет

- анализ частей речи (UPOS + XPOS);
- лемматизация;
- извлечение морфологических признаков (`Case`, `Gender`, `Animacy`, и т.д.);
- синтаксические зависимости и связь с главным словом;
- краткое контекстное объяснение роли токена в предложении;
- вывод в `json` или в читаемой табличной форме.

## Модели

По умолчанию используется **Stanza** (Stanford NLP) c моделями UD.  
Дополнительно можно выбрать `trankit` (если установлен) как альтернативную нейросетевую модель.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Примеры запуска

```bash
python pos_forensics.py --lang ru --text "Вчера на Невском проспекте мы быстро нашли старинную книгу." --format pretty
```

```bash
python pos_forensics.py --lang ru --backend stanza --input-file ./sample.txt --format json
```

## Загрузка моделей

При первом запуске для выбранного языка скрипт автоматически загрузит модель Stanza.

## Примечание

Для языков с особенно богатой морфологией (например, русский) рекомендуется использовать backend `stanza`, так как он лучше покрывает сочетание POS + морфологические признаки + зависимости.
